Objective

# 1 Descriptive Statistics
 Python Programming was used to verify the descriptive statistics across some important  key variables in the dataset.
# Key Findings
* The dataset contains 1,200 transactions.
* Customers purchase approximately 3 items per transaction.
* Average transaction value is approximately 1,054.
* Sales values show considerable variation across transactions.
# 2.Sales Trend Analysis
Monthly and quarterly sales trends were analyzed.
# Findings
* 2023 recorded the highest overall sales.
* 2025 recorded the lowest sales because only the first and second quarters were available.
* The first quarter generated the strongest sales in both 2023 and 2024.
* January consistently recorded the highest monthly sales.
* Lower sales were observed during later quarters in most years.
* # Correlation Analysis

# Key Insights
* Total Price has a strong positive relationship with Quantity.
* Total Price has a strong positive relationship with Unit Price.
* Quantity has almost no relationship with Unit Price.
* Items in Cart has a weak positive relationship with Total Price.
# Outlier Detection

* Eight high-value outliers were detected for Total Price, representing unusually expensive transactions.
